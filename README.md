# Thalita_projeto_Individual
Título: FASHION WORLD
Descrição: Um site desenvolvido para atender as funcionalidades de uma loja virtual especializada em vestidos de festa.
Funcionalidades: Na tela de LOGIN possibilita o usuário, que já possui uma conta, entrar na loja. A tela CADASTRO funciona para criar uma conta na loja. A Tela HOME permite a navegação do cliente nos produtos e a realização de compras. Na tela ENDEREÇO, o usuário que deseja realizar compras, realiza o cadastro de seu endereço para entrega de produtos. A tela DADOS DO USUÁRIO exbibe as informações registradas no cadastro (permitindo também a altereção de algum dado ou do endereço).
Tecnologias utilizadas: FlutterFlow, VS Code, Deploy, Render, Swagger e Java.
